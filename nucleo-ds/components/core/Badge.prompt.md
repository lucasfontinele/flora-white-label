Status pill for live order/document states. Pair `tone` with `dot` for lifecycle states.

```jsx
<Badge tone="warning" dot>Em análise</Badge>
<Badge tone="success" dot>Entregue</Badge>
<Badge tone="petrol">Rastreável</Badge>
```

Tones: `neutral | primary | accent | success | warning | error | info | petrol`. Sizes `sm | md`. Keep labels in the fixed order vocabulary (Solicitado → … → Entregue).
