Outline icon, wraps Lucide. Requires `<script src="https://unpkg.com/lucide@latest"></script>` on the page.

```jsx
<Icon name="package" size={20} />
<Icon name="shield-check" size={18} color="var(--color-primary)" />
```

Use kebab-case Lucide names. Default stroke 1.75 matches the brand's soft outline style. Never use cannabis-leaf icons; for product/strain use `flask-conical` or `sprout`.
